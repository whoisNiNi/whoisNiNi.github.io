水仙花数

打印水仙花数（3位水仙花、100-999之内的所有水仙花、三位及以上水仙花）
水仙花数是指一个3位数，它的每个位上的数字的3次幂之和等于它本身，
eg:

$$1^3·5^3·3^3=153$$

水仙花数
	public static void main(String[] args) {
		System.out.print("100-999之内的所有水仙花数:\t");
		for(int i=100; i<1000; i++){
			int b = i/100;
			int g = i/10%10;
			int s = i%10;
			if(Math.pow(b,3)+Math.pow(g,3)+Math.pow(s,3)==i)
			System.out.print(i+"\t");
		}
	}
//100-999之内的所有水仙花数:	153	370	371	407

排序

对（31, 42,21,50,12,60,81,74,101,93）进行排序（考点: java基础）。
程序分析:可以利用选择法，即从后9个比较过程中，选择一个最小的与第一个元素交换，
下次类推，即用第二个元素与后8个进行比较，并进行交换。

排序A
	public static void main(String[] args) {
		int[]  arrays = {31,42,21,50,12,60,81,74,101,93};
		int t = 0;
		for(int i = 0;i < arrays.length; i++) {
			for(int j = i; j < arrays.length;j++ ) {
				if(arrays[i]<arrays[j]) {
					t = arrays[j];
					arrays[j] = arrays[i];
					arrays[i] = t;
				}
			}
			
			if(i == arrays.length-1) {//判断:若是最后一位，则不用加大于号
				System.out.print(arrays[arrays.length-1]+" ");
			}else {
				System.out.print(arrays[i]+">"+"\t");
			}
		}
	}
// 101>	93>	81>	74>	60>	50>	42>	31>	21>	12

排序B
	public static void main(String[] args) {
		int[] arr = {31,42,21,50,12,60,81,74,101,93};
		int t = 0;
		for (int i = 0; i < arr.length; i++) {
			for (int j = 0; j < arr.length-i-1; j++) {
				if (arr[j]>arr[j+1]) {
					t = arr[j];
					arr[j] = arr[j+1];
					arr[j+1] = t;
				}
			}
			if(i == arr.length-1) {//判断:若是最后一位，则不用加大于号
				System.out.print(arr[arr.length-1]+" ");
			}else {
				System.out.print(arr[i]+"<"+"\t");
			}
		}
	}
// 31<	31<	31<	42<	50<	60<	74<	81<	93<	101
